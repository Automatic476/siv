# SIV

## Development

Install dependencies:

```bash
yarn
```

Start dev server:

```bash
yarn dev

# ready - started server on http://localhost:3000
```

## Layout

- http://siv.org - Landing page
- http://siv.org/admin - Create New Election
- http://siv.org/protocol - Protocol Description
- http://siv.org/faq - FAQ
